## Keyword Clustering Rules

**One primary keyword per page.** Never target the same keyword on two pages.

**Cluster mapping:**
- "html to image api" → Homepage (brand/product page)
- "html to image converter" → `/tools/html-to-image` (free tool)
- "html to image python" → `/docs/sdk/python` (SDK docs)
- "how to convert html to image" → `/blog/how-to-convert-html-to-image` (blog)

These are 4 different pages targeting 4 different intents, all in the same topic cluster.
