# Programmatic SEO Pages — HTMLPix.com

## Overview

pSEO pages are templated pages generated at scale from data. For HTMLPix, there are 6 categories of pages we can build, totaling 180-430 pages.

---

## 1. Free Tool Pages (`/tools/{name}`)

**Highest priority.** These are interactive tools that work in the browser and funnel users to the API.

### Page Template Structure

```
[H1: HTML to Image Converter Online]
[Subheading with keyword: Convert HTML/CSS to PNG, JPG, or WebP instantly]

[Interactive Tool]
- HTML input textarea (with example code pre-filled)
- CSS input textarea
- Format selector (PNG/JPG/WebP)
- Width/Height inputs
- "Convert" button
- Preview/download area

[How It Works section - 3 steps]
[FAQ section with FAQPage schema]
[CTA: "Need this at scale? Try the HTMLPix API →"]
[Related tools grid]
```

### Unique Value Per Page
- Each format page explains format-specific considerations (transparency for PNG, compression for JPG, modern browsers for WebP)
- Each tool page has unique FAQ content
- Tools actually work (powered by HTMLPix API on the backend)
- No login required to use (rate-limited for free)

1. Create 100+ starter templates across categories

3. Generate preview images for each template

### SEO Differentiator
- Each template is actual working HTML/CSS (not just a screenshot)
- Users can preview with their own variables
- One-click API integration
- Code is visible and copyable (search engines index it)

## Implementation Priority

### Phase 1 (Week 1-2): Technical Fixes + First Tools
- robots.txt, sitemap fixes, canonical tags, JSON-LD
- Build `/tools/html-to-image` (the #1 traffic page)
- Build `/tools/og-image-generator`

### Phase 2 (Week 3-4): Templates + Use Cases
- Create 20-30 OG image starter templates
- Build template gallery pages with public URLs
- Build top 5 use case pages

### Phase 3 (Month 2): Comparisons + Integrations
- Build all comparison/alternative pages
- Build top 6 integration pages (Next.js, Python, Node.js, Go, PHP, Ruby)
- First 3 blog articles

### Phase 4 (Month 3+): Scale
- Expand template gallery to 100+ templates
- Build remaining integration pages
- Publish 2-3 blog articles per month
- Add remaining use case pages
- Build additional free tools
