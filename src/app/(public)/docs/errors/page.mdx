# Errors

All errors return JSON:

```json
{
  "code": "INVALID_SIGNATURE",
  "message": "Invalid signed image URL"
}
```

## POST /v1/image-url

| HTTP | Code | Meaning |
|------|------|---------|
| 400 | `INVALID_JSON` | Body is not valid JSON |
| 400 | `MISSING_TEMPLATE_ID` | `templateId` missing |
| 400 | `INVALID_VARIABLES` | `variables` must be an object |
| 400 | `INVALID_WIDTH` / `INVALID_HEIGHT` | Size outside 1-4096 |
| 400 | `INVALID_FORMAT` | Unsupported image format |
| 401 | `MISSING_KEY` / `INVALID_KEY` | API key issue |
| 403 | `KEY_INACTIVE` | API key revoked |
| 402 | `SUBSCRIPTION_INACTIVE` | Subscription inactive |
| 429 | `QUOTA_EXCEEDED` | Monthly quota exhausted |
| 404 | `TEMPLATE_NOT_FOUND` | Template missing or not accessible |

## GET /v1/image

| HTTP | Code | Meaning |
|------|------|---------|
| 400 | `MISSING_TEMPLATE_ID` / `MISSING_UID` / `MISSING_EXP` / `MISSING_SIG` | Required signed params missing |
| 400 | `MISSING_TEMPLATE_VARIABLES` | Required template variable missing |
| 403 | `URL_EXPIRED` | Signed URL expired |
| 403 | `INVALID_SIGNATURE` | Signature mismatch |
| 404 | `TEMPLATE_NOT_FOUND` | Template missing or not accessible |
| 429 | `QUOTA_EXCEEDED` | User over quota on uncached render |
| 500 | `RENDER_ERROR` | Takumi render failure |

## Deprecated endpoints

| HTTP | Code | Meaning |
|------|------|---------|
| 410 | `ENDPOINT_DEPRECATED` | `/render` or `/images/:id` is deprecated |
