# Quick Start

HTMLPix is now OG-first:

1. Call `POST /v1/image-url` with your API key and template inputs.
2. Receive a signed public URL.
3. Use that URL in `<meta property=\"og:image\">`.

## 1) Mint a signed image URL

```bash
curl -X POST https://api.htmlpix.com/v1/image-url \
  -H "Authorization: Bearer hpx_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "templateId": "k57k8w9n7n9r8b6r2x6f2q9v1h7p4y0c",
    "width": 1200,
    "height": 630,
    "format": "png",
    "variables": {
      "title": "Launch Day",
      "subtitle": "OG image generated by HTMLPix"
    }
  }'
```

Response:

```json
{
  "url": "https://api.htmlpix.com/v1/image?templateId=...&uid=...&exp=...&v_title=Launch%20Day&sig=...",
  "expiresAt": 1914998400000
}
```

## 2) Use it in OG tags

```html
<meta property="og:image" content="https://api.htmlpix.com/v1/image?...&sig=..." />
<meta name="twitter:image" content="https://api.htmlpix.com/v1/image?...&sig=..." />
```

## 3) Keep API keys private

Never call `POST /v1/image-url` from browser code. Mint URLs from your backend/server actions only.
